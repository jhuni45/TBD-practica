# Framework Editor de Imágenes

## Requerimientos o dependencias

```bash
c++11
OpenCV (>= 3.0)
QtCreator (>= 4.x.x)
```
## Contenido del Framework
 - Operaciones elementales
 - Ecualizador (histograma)
 - Transformaciones
 - Filtros
 - Convoluciones
 - Combinación de imágenes
