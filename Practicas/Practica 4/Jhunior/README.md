## Ejecución
Compilación
```bash
# Creamos el makefile que enlace las librerias
cmake .
# Compilamos el código
make
# Ejecutamos
./DisplayImage <imagen1> <imagen2>
# Ejemplos
./DisplayImage pokachu.png pokachu.png
```
